## Development
You can use a Makefile to simplify your development environment. To start the development process, run the following command:
```bash
make dev
```

### For windows users
**1. Install chocolatey**  
Follow the instructions on the
[Chocolatey installation page](https://chocolatey.org/install)

**2. Verify installation**  
Ensure Chocolatey is installed correctly by running:
```bash
choco --version
```

**3. Install make with chocolatey**  
Install Make by running:
```bash
choco install make
```

**4. Verify installation**:  
Check if Make is installed correctly with:
```bash
make --version
```
